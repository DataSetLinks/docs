Getting an API token is the first step toward connecting your systems and applications to DataLinks. Your token lets you authenticate securely, access your workspace, and begin sending data to be structured, linked, and enriched.

Follow the steps below to find your token and how to copy it safely so you can start integrating with the platform.

## How to create a new token

1. Log in to the DataLinks platform.
2. Click **Settings** in the left navigation menu.
3. Click the **Tokens & API** tab.
3. Click **\+ Add a new** token.
4. Enter your desired token name and set an expiration date (for example, one month).

   <Warning>
     Your connected app will stop working when the token expires. Make sure you have enough time to renew it or plan to update the token details before the expiry date.
   </Warning>
5. Click **\+ Create token**.
6. Copy and save your token.

   <Info>
     You can only view your token information during creation. Save a copy in a secure place so you can access it later. Otherwise, you will need to create a new token.
